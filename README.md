**Introverted Personality Analysis**

A Machine Learning project that analyzes personality traits and predicts whether a person is Introverted or Extroverted based on behavioral and social interaction features.

**📌 Project Overview**

This project uses classification algorithms to identify personality type from structured data such as:

Social interaction frequency

Public speaking comfort level

Social media usage

Group activity preference

Energy level after social events

The goal is to demonstrate data preprocessing, feature engineering, model building, and evaluation in a real-world psychological dataset.
